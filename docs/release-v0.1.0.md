# Release Draft: v0.1.0

## Summary

Initial public release of `inventory-sync`.

This version publishes the current self-hosted BDS inventory sync workflow as a portfolio-ready repository.

## Included

- TypeScript Behavior Pack source for BDS
- Node.js / Express VPS API
- snapshot save / load / loadbackup flow
- automatic pre-load backup
- machine-readable and readable audit logs
- duplication prevention through post-save clearing and single-use loads
- bundle preparation script for BDS deployment

## Notes

- This repository is sanitized for GitHub and uses placeholder configuration values
- Portable storage items such as shulker boxes are still excluded on the current runtime
- The project is experimental and focused on a practical self-hosted setup

## Suggested Release Title

`v0.1.0 - Initial public inventory sync release`
